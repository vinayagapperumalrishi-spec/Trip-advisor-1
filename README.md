# 🌍 TripAdvisor Clone (HTML & CSS)

This project is a detailed front-end clone of the TripAdvisor website, built entirely using HTML and CSS. It is designed to replicate the visual structure and user interface of a modern travel platform while maintaining clean code practices and a strong focus on design accuracy.

The application includes two fully structured pages: a **Home Page**, which introduces users to trending destinations, travel categories, and featured sections, and an **Explore Page**, which provides a deeper look into travel options with organized layouts and content sections. Both pages are carefully designed to ensure consistency, readability, and an engaging browsing experience.

A major highlight of this project is the emphasis on **layout design and responsiveness**. Using CSS Flexbox and Grid, the interface adapts smoothly across different screen sizes, ensuring accessibility on desktops, tablets, and mobile devices. Attention has been given to spacing, typography, color combinations, and alignment to closely mimic a professional travel website.

The project also demonstrates the ability to break down a complex UI into manageable sections such as navigation bars, cards, image layouts, and content containers. Each component is structured in a reusable and scalable way, making the code easy to maintain and extend in the future.

Unlike dynamic applications, this clone is built without JavaScript or external frameworks. This approach highlights strong foundational skills in HTML and CSS, proving that visually rich and responsive designs can be achieved with core web technologies alone.
